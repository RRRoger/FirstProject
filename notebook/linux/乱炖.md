### 2.JavaScript 时间处理的函数

```javascript
/**
* 字符串转时间(`yyyy-MM-dd HH:mm:ss` | `yyyy-MM-dd`)
* 暂时只满足以上两种格式
*/
String.prototype.rogerStringToDate=function(){
    t = this.split(/[-\ :]/);
    t[1]--;
    return new Date(...t);
};
```
